Go & Rust is a modern, compiled, open source programming language designed for high performance systems where speed & memory management is their priority. Go (by google) for concurrency & Rust is famous for their memory safety.

Webstack📚

  1. HTML: HTML(Hyper Text Markup Language) is the basic structure of a webpage. HTML is like a skeleton🩻.
  2. CSS: CSS(Cascading Style Sheet) give style to the webpage by adding padding, colours, text design, attractive buttons▶️.
  3. JavaScript (JS): JS is a programming language for the web. JS can calculate, manipulate, validate data. In simple language it is the brain of a website.
  4. TypeScript: TS is a popular open source programming language developed by Microsoft. It allows developers to catch errors early during the development through the type checking before the code is converted into plain JS. It is highly suitable for large applications.
Webstack is used in
  • Frontend Developer
  • UI/UX Implementation
  • Full Stack Developer
